## 🔧 第二步：部署服务器环境

### 2.1 上传部署脚本

**方法一：使用 Git（推荐）**
```bash
# 在服务器上执行
cd /root
git clone https://github.com/dandan812/wallpaper-uniapp.git
cd wallpaper-uniapp/scripts
chmod +x *.sh
```

**方法二：手动上传**
```bash
# 在本地执行（需要安装 scp）
scp scripts/*.sh root@8.135.46.112:/root/
```

### 2.2 运行环境部署脚本

```bash
# 在服务器上执行
cd /root/wallpaper-uniapp/scripts
./deploy-server.sh
```

**脚本会自动安装:**
- ✅ Node.js 18.x
- ✅ MySQL 8.0
- ✅ Redis 6+
- ✅ Nginx
- ✅ PM2

**安装过程中需要输入:**
1. MySQL root 密码（请记住！）
2. 数据库名称（默认: wallpaper）
3. 数据库用户名（默认: wallpaper_user）
4. 数据库用户密码（请记住！）
5. Redis 密码（请记住！）

**预计时间:** 10-15 分钟

## 📦 第四步：创建后端项目

### 4.1 创建项目目录

```bash
# 在服务器上执行
mkdir -p /var/www/wallpaper-api
cd /var/www/wallpaper-api
```

### 4.2 初始化 Node.js 项目

```bash
npm init -y
```

### 4.3 安装依赖

```bash
npm install express sequelize mysql2 redis ioredis
npm install dotenv cors helmet compression
npm install morgan winston express-rate-limit
npm install ali-oss
npm install xss
```

**预计时间:** 3-5 分钟

---

## ⚙️ 第五步：创建配置文件

### 5.1 创建 .env 文件

```bash
# 在服务器上执行
cd /var/www/wallpaper-api
nano .env
```

**复制以下内容（修改密码）:**
```env
# 环境
NODE_ENV=production
PORT=3000

# 数据库（从 /root/wallpaper-config.txt 获取）
DB_HOST=localhost
DB_PORT=3306
DB_NAME=wallpaper
DB_USER=wallpaper_user
DB_PASSWORD=你的数据库密码

# Redis（从 /root/wallpaper-config.txt 获取）
REDIS_HOST=localhost
REDIS_PORT=6379
REDIS_PASSWORD=你的Redis密码

# API密钥（自己设置）
ACCESS_KEYS=key123456,key789012

# OSS（暂时留空，后续配置）
OSS_REGION=
OSS_ACCESS_KEY_ID=
OSS_ACCESS_KEY_SECRET=
OSS_BUCKET=
```

**保存:** Ctrl+O, 回车, Ctrl+X

### 5.2 查看配置信息

```bash
# 查看之前保存的配置
cat /root/wallpaper-config.txt
```

## ⚠️ 常见问题

### 问题1: MySQL 连接失败
```bash
# 检查 MySQL 是否运行
systemctl status mysql

# 重启 MySQL
systemctl restart mysql
```

### 问题2: Redis 连接失败
```bash
# 检查 Redis 是否运行
systemctl status redis-server

# 重启 Redis
systemctl restart redis-server
```

### 问题3: 端口被占用
```bash
# 查看端口占用
netstat -tulpn | grep :3000
netstat -tulpn | grep :3306
netstat -tulpn | grep :6379
```
