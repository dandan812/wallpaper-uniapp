# 前端部署 - 服务器端配置命令

> 归档说明：这是旧版服务器配置命令，当前部署方式请优先参考 `docs/deployment/前端H5部署.md`。

直接在服务器上执行以下命令:

```bash
# 1. 创建部署目录
mkdir -p /var/www/wallpaper-h5

# 2. 配置 Nginx
cat > /etc/nginx/sites-available/wallpaper-h5 << 'EOF'
server {
    listen 80;
    server_name 8.135.46.112;

    root /var/www/wallpaper-h5;
    index index.html;

    # H5 前端
    location / {
        try_files $uri $uri/ /index.html;
    }

    # API 代理
    location /api/ {
        proxy_pass http://localhost:3000/api/;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_cache_bypass $http_upgrade;
    }

    # 静态资源缓存
    location ~* \.(jpg|jpeg|png|gif|ico|css|js|woff|woff2|ttf)$ {
        expires 30d;
        add_header Cache-Control "public, immutable";
    }

    # Gzip 压缩
    gzip on;
    gzip_vary on;
    gzip_min_length 1024;
    gzip_types text/plain text/css text/xml text/javascript application/x-javascript application/xml+rss application/json;
}
EOF

# 3. 启用配置
ln -sf /etc/nginx/sites-available/wallpaper-h5 /etc/nginx/sites-enabled/

# 4. 测试 Nginx 配置
nginx -t

# 5. 重载 Nginx
systemctl reload nginx

# 6. 检查状态
echo ""
echo "✓ Nginx 配置完成"
echo "✓ 部署目录: /var/www/wallpaper-h5"
echo ""
echo "下一步: 上传前端文件到 /var/www/wallpaper-h5/"
```

## 完整执行步骤

### 在服务器上执行:

```bash
# 复制以下所有命令,一次性粘贴执行

mkdir -p /var/www/wallpaper-h5

cat > /etc/nginx/sites-available/wallpaper-h5 << 'EOF'
server {
    listen 80;
    server_name 8.135.46.112;
    root /var/www/wallpaper-h5;
    index index.html;
    location / {
        try_files $uri $uri/ /index.html;
    }
    location /api/ {
        proxy_pass http://localhost:3000/api/;
        proxy_http_version 1.1;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    }
    location ~* \.(jpg|jpeg|png|gif|ico|css|js|woff|woff2|ttf)$ {
        expires 30d;
    }
    gzip on;
    gzip_types text/plain text/css application/json application/javascript;
}
EOF

ln -sf /etc/nginx/sites-available/wallpaper-h5 /etc/nginx/sites-enabled/
nginx -t && systemctl reload nginx

echo "✓ 服务器配置完成!"
echo "✓ 部署目录: /var/www/wallpaper-h5"
```

### 在本地执行 (上传文件):

```bash
# 方法1: 使用 SCP (需要先在 HBuilderX 编译)
cd unpackage/dist/build/h5
scp -r * root@8.135.46.112:/var/www/wallpaper-h5/

# 方法2: 使用 Git (如果代码在仓库)
# 在服务器执行:
cd /var/www/wallpaper-h5
git clone <仓库地址> .
```
