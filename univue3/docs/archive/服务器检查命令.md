# 服务器检查命令

> 归档说明：这是一次性排查命令记录，当前请结合 `docs/deployment/前端H5部署.md` 使用。

请在服务器上执行以下命令检查：

```bash
# 1. 检查文件是否上传成功
ls -la /var/www/wallpaper-h5/

# 2. 检查 Nginx 状态
systemctl status nginx

# 3. 检查 Nginx 配置
nginx -t

# 4. 查看 Nginx 错误日志
tail -20 /var/log/nginx/error.log

# 5. 检查端口占用
netstat -tulpn | grep :80

# 6. 测试本地访问
curl -I http://localhost
```

请把这些命令的输出结果发给我，我帮你诊断问题。
