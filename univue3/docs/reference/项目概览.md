# 项目概览

## 1. 项目组成

当前仓库包含两部分：

- 前端：`univue3/`
- 后端：`backend-code/`

前端使用 `uni-app + Vue 3`，后端使用 `Node.js + Express + Sequelize`。

## 2. 当前目录结构

```text
uniappvue3/
├── univue3/              # 前端项目
│   ├── api/
│   ├── components/
│   ├── config/
│   ├── docs/
│   ├── pages/
│   ├── static/
│   └── utils/
└── backend-code/         # 后端项目
    ├── src/
    ├── scripts/
    ├── .env.example
    └── package.json
```

## 3. 技术栈

### 前端

- uni-app
- Vue 3
- HBuilderX

### 后端

- Node.js
- Express
- Sequelize
- MySQL
- Redis
- PM2
- Nginx

## 4. 当前主接口

前后端统一后的主路径为：

- `GET /api/categories`
- `GET /api/banners`
- `GET /api/notices`
- `GET /api/notices/:id`
- `GET /api/wallpapers`
- `GET /api/wallpapers/:id`
- `GET /api/wallpapers/random`
- `GET /api/wallpapers/search`
- `GET /api/users/me`
- `GET /api/users/wallpapers`
- `POST /api/wallpapers/score`
- `POST /api/wallpapers/download`

后端仍保留旧路径兼容，但主文档和前端代码都已经切到这套新命名。

## 5. 当前数据状态

当前线上与本地已经对齐：

- 分类数：`12`
- 壁纸数：`360`
- 每个分类 `wallpaper_count`：`30`

## 6. 运行方式

### 前端

前端通常通过 HBuilderX 运行或编译：

- H5 构建目录：`unpackage/dist/build/web`
- 微信小程序构建目录：`unpackage/dist/dev/mp-weixin` 或 `unpackage/dist/build/mp-weixin`

### 后端

```bash
cd ../backend-code
npm install
npm run dev
```
