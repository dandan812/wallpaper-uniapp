# 壁纸小程序后端开发规划文档

## 项目概述

**项目名称**: 壁纸小程序后端 API 服务  
**技术栈**: Node.js + Express + MySQL + Redis + 阿里云OSS  
**服务器配置**: 2核2G 云服务器  
**开发周期**: 14-21 天  
**文档版本**: v1.0  
**创建日期**: 2026-03-18

---

## 技术架构

### 后端技术栈
- **运行环境**: Node.js 18+
- **Web框架**: Express 4.x
- **数据库**: MySQL 8.0+
- **ORM**: Sequelize 6.x
- **缓存**: Redis 6+
- **图片存储**: 阿里云 OSS / 腾讯云 COS
- **进程管理**: PM2
- **反向代理**: Nginx

### 服务器配置优化（2核2G）
```
- MySQL: 512MB - 768MB
- Redis: 128MB - 256MB
- Node.js: 512MB - 768MB
- 系统预留: 256MB - 512MB
```

## 数据库设计

### 1. 壁纸表 (wallpapers)
```sql
CREATE TABLE `wallpapers` (
  `id` INT PRIMARY KEY AUTO_INCREMENT,
  `classid` INT NOT NULL COMMENT '分类ID',
  `title` VARCHAR(200) DEFAULT '' COMMENT '标题',
  `description` TEXT COMMENT '描述',
  `picurl` VARCHAR(500) NOT NULL COMMENT '原图URL',
  `small_picurl` VARCHAR(500) NOT NULL COMMENT '缩略图URL',
  `tabs` JSON COMMENT '标签数组',
  `score` DECIMAL(2,1) DEFAULT 0 COMMENT '平均评分',
  `score_count` INT DEFAULT 0 COMMENT '评分人数',
  `download_count` INT DEFAULT 0 COMMENT '下载次数',
  `view_count` INT DEFAULT 0 COMMENT '浏览次数',
  `nickname` VARCHAR(100) DEFAULT '' COMMENT '发布者昵称',
  `status` TINYINT DEFAULT 1 COMMENT '状态: 0-待审核, 1-已发布, 2-已下架',
  `created_at` TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  `updated_at` TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
  INDEX idx_classid_status (classid, status, created_at),
  INDEX idx_status (status)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='壁纸表';
```

### 2. 分类表 (categories)
```sql
CREATE TABLE `categories` (
  `id` INT PRIMARY KEY AUTO_INCREMENT,
  `name` VARCHAR(50) NOT NULL COMMENT '分类名称',
  `picurl` VARCHAR(500) DEFAULT '' COMMENT '分类封面图',
  `sort` INT DEFAULT 0 COMMENT '排序',
  `select` TINYINT DEFAULT 0 COMMENT '是否推荐: 0-否, 1-是',
  `wallpaper_count` INT DEFAULT 0 COMMENT '壁纸数量',
  `created_at` TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  `updated_at` TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
  INDEX idx_select_sort (select, sort)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='分类表';
```

### 3. 轮播图表 (banners)
```sql
CREATE TABLE `banners` (
  `id` INT PRIMARY KEY AUTO_INCREMENT,
  `picurl` VARCHAR(500) NOT NULL COMMENT '图片URL',
  `url` VARCHAR(500) DEFAULT '' COMMENT '跳转链接',
  `target` VARCHAR(20) DEFAULT 'self' COMMENT '跳转方式: self/miniProgram',
  `appid` VARCHAR(50) DEFAULT '' COMMENT '小程序appid',
  `sort` INT DEFAULT 0 COMMENT '排序',
  `status` TINYINT DEFAULT 1 COMMENT '状态: 0-禁用, 1-启用',
  `created_at` TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  INDEX idx_status_sort (status, sort)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='轮播图表';
```

### 4. 用户表 (users)
```sql
CREATE TABLE `users` (
  `id` INT PRIMARY KEY AUTO_INCREMENT,
  `openid` VARCHAR(100) UNIQUE COMMENT '微信openid',
  `nickname` VARCHAR(100) DEFAULT '' COMMENT '昵称',
  `avatar` VARCHAR(500) DEFAULT '' COMMENT '头像',
  `ip` VARCHAR(50) DEFAULT '' COMMENT 'IP地址',
  `address` VARCHAR(200) DEFAULT '' COMMENT '地址信息',
  `created_at` TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  `last_login_at` TIMESTAMP NULL COMMENT '最后登录时间',
  INDEX idx_openid (openid)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='用户表';
```

### 5. 评分记录表 (scores)
```sql
CREATE TABLE `scores` (
  `id` INT PRIMARY KEY AUTO_INCREMENT,
  `user_id` INT NOT NULL COMMENT '用户ID',
  `wallpaper_id` INT NOT NULL COMMENT '壁纸ID',
  `classid` INT NOT NULL COMMENT '分类ID',
  `score` TINYINT NOT NULL COMMENT '评分 1-5',
  `created_at` TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  UNIQUE KEY uk_user_wallpaper (user_id, wallpaper_id),
  INDEX idx_wallpaper (wallpaper_id),
  INDEX idx_user_created (user_id, created_at)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='评分记录表';
```

### 6. 下载记录表 (downloads)
```sql
CREATE TABLE `downloads` (
  `id` INT PRIMARY KEY AUTO_INCREMENT,
  `user_id` INT NOT NULL COMMENT '用户ID',
  `wallpaper_id` INT NOT NULL COMMENT '壁纸ID',
  `classid` INT NOT NULL COMMENT '分类ID',
  `created_at` TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  INDEX idx_user_created (user_id, created_at),
  INDEX idx_wallpaper (wallpaper_id)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='下载记录表';
```

### 7. 公告表 (notices)
```sql
CREATE TABLE `notices` (
  `id` INT PRIMARY KEY AUTO_INCREMENT,
  `title` VARCHAR(200) NOT NULL COMMENT '标题',
  `content` TEXT COMMENT '内容（富文本）',
  `author` VARCHAR(100) DEFAULT '' COMMENT '作者',
  `view_count` INT DEFAULT 0 COMMENT '阅读量',
  `select` TINYINT DEFAULT 0 COMMENT '是否置顶: 0-否, 1-是',
  `status` TINYINT DEFAULT 1 COMMENT '状态: 0-草稿, 1-已发布',
  `created_at` TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  `updated_at` TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
  INDEX idx_select_status (select, status, created_at)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='公告表';
```

## API 接口设计

### 统一响应格式
```json
{
  "errCode": 0,
  "errMsg": "查询成功",
  "data": {},
  "total": 100,
  "timeCost": 50
}
```

### 接口列表

#### 1. 首页海报 - GET /api/homeBanner
- **功能**: 获取轮播图列表
- **参数**: 无
- **缓存**: 10分钟

#### 2. 壁纸大分类 - GET /api/classify
- **功能**: 获取分类列表
- **参数**: pageNum, pageSize, select
- **缓存**: 10分钟

#### 3. 分类壁纸列表 - GET /api/wallList
- **功能**: 获取某分类下的壁纸列表
- **参数**: classid(必填), pageNum, pageSize
- **缓存**: 3分钟

#### 4. 随机9张壁纸 - GET /api/randomWall
- **功能**: 首页每日推荐
- **参数**: 无
- **缓存**: 5分钟

#### 5. 壁纸详情 - GET /api/detailWall
- **功能**: 获取单个壁纸详细信息
- **参数**: id(必填)
- **副作用**: 浏览量+1

#### 6. 为壁纸评分 - POST /api/setupScore
- **功能**: 用户对壁纸评分
- **参数**: classid, wallId, userScore, userId

#### 7. 壁纸下载 - POST /api/downloadWall
- **功能**: 记录下载行为
- **参数**: classid, wallId, userId

#### 8. 用户历史列表 - GET /api/userWallList
- **功能**: 获取用户的评分/下载历史
- **参数**: type(score/download), pageNum, pageSize, userId

#### 9. 用户信息 - GET /api/userInfo
- **功能**: 获取用户基本信息
- **参数**: userId

#### 10. 公告列表 - GET /api/wallNewsList
- **功能**: 获取公告列表
- **参数**: select, pageNum, pageSize

#### 11. 公告详情 - GET /api/wallNewsDetail
- **功能**: 获取公告详细内容
- **参数**: id(必填)
- **副作用**: 阅读量+1

#### 12. 搜索壁纸 - GET /api/searchWall
- **功能**: 关键词搜索壁纸
- **参数**: keyword, pageNum, pageSize

## 部署方案

### 服务器环境
- Ubuntu 20.04 / CentOS 7+
- Node.js 18+
- MySQL 8.0
- Redis 6+
- Nginx

### 部署步骤
```bash
# 1. 安装依赖
npm install --production

# 2. 初始化数据库
npm run init-db

# 3. 使用 PM2 启动
pm2 start ecosystem.config.js

# 4. 配置 Nginx
# /etc/nginx/sites-available/wallpaper-api
server {
    listen 80;
    server_name api.yourdomain.com;
    
    location / {
        proxy_pass http://localhost:3000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}

# 5. 配置 HTTPS
certbot --nginx -d api.yourdomain.com
```

## 风险提示

1. **版权问题**: 壁纸来源需合法
2. **内容审核**: 需要实现内容审核机制
3. **性能瓶颈**: 图片加载必须使用 CDN

5. **安全防护**: 防止 SQL 注入、XSS、DDoS 攻击
